template<typename Op >
py::object bitwise_core ( py::array  z2r_1,
py::array  z2r_2,
Op  op 
)

This templated function performs an element-wise, bitwise operation onto two NumPy contiguous (C-like) arrays.

Any other type of operators or type of arrays will lead to undefined behavior. This function is the basis all two-array bitwise operations.

Template Parameters
OpThe type of the bitwise operator (std::bit_and<uint64_t>, std::bit_xor<uint64_t>...)
Parameters
z2r_1The first input array from Python.
z2r_2The second input array from Python
opThe bitwise operator to apply
Returns
py::array A NumPy contiguous array of the same shape and dtype as the inputs, containing the result of the operation
